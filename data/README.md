# Test Datasets and Image Sources

This README file provides an overview of the test datasets and image sources used in the Mars Feature Detection project.

## Test Datasets

1. **NASA Mars Reconnaissance Orbiter Dataset**  
   - Description: Contains high-resolution images of Mars taken by the HiRISE camera on the Mars Reconnaissance Orbiter.
   - Source: [NASA Planetary Data System](https://pds.nasa.gov/)

2. **Mars Global Surveyor Dataset**  
   - Description: Includes images and data from the Mars Global Surveyor (MGS) mission, providing various features observed on the Mars surface.
   - Source: [NASA JPL](https://www.jpl.nasa.gov)

3. **Mars Exploration Rover Dataset**  
   - Description: Provides images taken by the Spirit and Opportunity rovers, showcasing the Martian landscape and surface features.
   - Source: [NASA Jet Propulsion Laboratory](https://www.jpl.nasa.gov/)

## Image Sources

- All images utilized in the project are obtained from official NASA repositories to ensure reliability and accuracy.
